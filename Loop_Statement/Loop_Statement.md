## 1. for 문

> 횟수가 **정해져 있을 때** 사용!

### 구조
```c
for (시작; 조건; 증가) {
    // 반복할 코드
}
```

### 예시
```c
for (int i = 1; i <= 4; i++) {
    printf("%d ", i);
}
// 출력: 1 2 3 4
```

### 흐름
```
 i  | i<=4? | 출력
----+-------+-----
 1  |  ✅   |  1
 2  |  ✅   |  2
 3  |  ✅   |  3
 4  |  ✅   |  4
 5  |  ❌   |  탈출
```

## 2. while 문

> 조건이 **참인 동안** 계속 반복!

### 구조
```c
while (조건) {
    // 반복할 코드
}
```

### 예시
```c
int i = 1;
while (i <= 5) {
    printf("%d ", i);
    i++;
}
// 출력: 1 2 3 4 5
```

### 흐름
```
 i  | i<=5? | 출력 | i++
----+-------+------+----
 1  |  ✅   |  1   | i→2
 2  |  ✅   |  2   | i→3
 3  |  ✅   |  3   | i→4
 4  |  ✅   |  4   | i→5
 5  |  ✅   |  5   | i→6
 6  |  ❌   | 탈출
```

> ⚠️ **주의:** 조건이 항상 참이면 무한루프 발생!
> ```c
> while (1 <= 5) { ... }  // 1<=5 는 항상 true → 무한루프 💀
> while (i <= 5) { ... }  // i 가 변하므로 정상 종료 ✅
> ```

## 3. do~while 문

> **무조건 1번은 실행**, 그 다음에 조건 확인!

### 구조
```c
do {
    // 반복할 코드
} while (조건);
```

### 예시
```c
int x = 0;
do {
    printf("x = %d\n", x);
    x--;
} while (x > 0);
// 출력: x = 0   ← x=0 이어도 1번은 무조건 실행!
```

### while vs do~while 차이
```
[ while ]                     [ do~while ]
조건 확인 → 실행              실행 → 조건 확인

x=0 이면?                     x=0 이면?
0>0 ❌ → 아무것도 출력 안 됨   일단 실행 → x=0 출력 → 0>0 ❌ 탈출
```

## 4. break

> 루프를 **즉시 완전 탈출!**

### 구조
```c
if (조건) {
    break;  // 여기서 즉시 루프 탈출
}
```

### 예시
```c
for (int i = 1; i <= 10; i++) {
    if (i > 4) {
        break;      // i가 5 되는 순간 즉시 탈출!
    }
    printf("%d ", i);
}
// 출력: 1 2 3 4
```

### 흐름
```
 i  | i>4? | 출력
----+------+-----
 1  |  ❌  |  1
 2  |  ❌  |  2
 3  |  ❌  |  3
 4  |  ❌  |  4
 5  |  ✅  |  break! → 즉시 탈출 (출력 없음)
```

> ⚠️ **핵심:** break 아래 코드는 **절대 실행 안 됨!**

## 5. continue

> 이번 회차만 **건너뛰고** 다시 위로!

### 구조
```c
if (조건) {
    continue;  // 아래 코드 건너뛰고 다시 반복 시작
}
```

### 예시
```c
for (int i = 1; i <= 7; i++) {
    if (i % 2 == 0) {
        continue;   // 짝수면 출력 건너뜀
    }
    printf("%d ", i);
}
// 출력: 1 3 5 7
```

### 흐름
```
 i  | i%2==0? | 동작
----+---------+---------------------------
 1  |  1%2=1 ❌ | printf → 1 출력
 2  |  2%2=0 ✅ | continue → printf 건너뜀
 3  |  3%2=1 ❌ | printf → 3 출력
 4  |  4%2=0 ✅ | continue → printf 건너뜀
 5  |  5%2=1 ❌ | printf → 5 출력
 6  |  6%2=0 ✅ | continue → printf 건너뜀
 7  |  7%2=1 ❌ | printf → 7 출력
```

> 💡 **% 기호란?** 나머지를 구하는 기호!
> - `7 % 2 = 1` → 홀수
> - `6 % 2 = 0` → 짝수

## 🔑 최종 정리

| 키워드 | 한 줄 요약 |
|--------|-----------|
| `for` | 횟수가 정해졌을 때 |
| `while` | 조건이 참인 동안 반복 |
| `do~while` | 무조건 1번 실행 후 조건 확인 |
| `break` | 루프 즉시 완전 탈출 |
| `continue` | 이번 회차만 건너뛰고 계속 |

## 💡 전위/후위 연산자 보너스

| 기호 | 뜻 |
|------|----|
| `i++` | 출력 먼저 → 나중에 올려 |
| `++i` | 먼저 올리고 → 출력 |
| `i--` | 출력 먼저 → 나중에 깎아 |
| `--i` | 먼저 깎고 → 출력 |
