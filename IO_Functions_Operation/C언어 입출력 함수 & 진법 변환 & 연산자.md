# C언어 입출력 함수 & 진법 변환 & 연산자

---

## 1. 입출력 함수 (I/O Functions)

### (1) 표준 출력 함수

데이터를 **화면(모니터)에 출력**하는 함수.

| 함수 | 설명 |
|------|------|
| `printf()` | 형식 지정자를 이용해 다양한 자료형 출력 |
| `putchar()` | 문자 하나 출력 |
| `puts()` | 문자열 출력 (자동 줄바꿈 포함) |

---

### (2) 표준 입력 함수

키보드로부터 **데이터를 입력**받는 함수.

| 함수 | 설명 |
|------|------|
| `scanf()` | 형식 지정자를 이용해 다양한 자료형 입력 |
| `getchar()` | 문자 하나 입력 |
| `gets()` | 문자열 입력 (보안 취약점으로 사용 비권장) |

---

### (3) 출력 형식 지정자 (Format Specifier)

`printf()` / `scanf()` 에서 자료형을 지정할 때 사용.

| 지정자 | 자료형 | 예시 |
|--------|--------|------|
| `%d` | 정수 (int) | `printf("%d", 10)` → `10` |
| `%f` | 실수 (float) | `printf("%f", 3.14)` → `3.140000` |
| `%lf` | 실수 (double) | `scanf("%lf", &x)` |
| `%c` | 문자 (char) | `printf("%c", 'A')` → `A` |
| `%s` | 문자열 (string) | `printf("%s", "hello")` → `hello` |
| `%o` | 8진수 | `printf("%o", 8)` → `10` |
| `%x` | 16진수 (소문자) | `printf("%x", 255)` → `ff` |
| `%X` | 16진수 (대문자) | `printf("%X", 255)` → `FF` |

---

### (4) 이스케이프 문자 (Escape Character)

`\`로 시작하는 특수 문자. 화면에 직접 출력할 수 없는 동작을 표현.

| 이스케이프 문자 | 의미 |
|----------------|------|
| `\n` | 줄바꿈 (newline) |
| `\t` | 탭 (tab) |
| `\\` | 백슬래시 `\` 출력 |
| `\"` | 큰따옴표 `"` 출력 |
| `\'` | 작은따옴표 `'` 출력 |
| `\0` | 널 문자 (문자열 끝) |
| `\r` | 커서를 줄 맨 앞으로 이동 |
| `\a` | 경고음 (beep) |

---

### (5) 표준 입출력 함수 사용법

#### 1) printf() / scanf()

```c
#include <stdio.h>

int main() {
    int age;
    printf("나이를 입력하세요: ");
    scanf("%d", &age);          // &: 주소 연산자 (변수의 주소에 저장)
    printf("나이: %d세\n", age);
    return 0;
}
```

> `scanf()`에서 변수 앞에 `&`를 붙여야 한다. (문자열 `%s`는 예외)

---

#### 2) getchar() / putchar()

문자 **하나씩** 입출력할 때 사용.

```c
#include <stdio.h>

int main() {
    char ch;
    printf("문자 입력: ");
    ch = getchar();     // 문자 하나 입력
    putchar(ch);        // 문자 하나 출력
    putchar('\n');
    return 0;
}
```

---

#### 3) gets() / puts()

문자열 입출력. 공백 포함 한 줄 전체를 입력받을 수 있다.

```c
#include <stdio.h>

int main() {
    char name[50];
    printf("이름 입력: ");
    gets(name);     // 공백 포함 입력 (보안상 fgets() 권장)
    puts(name);     // 문자열 출력 + 자동 줄바꿈
    return 0;
}
```

> `gets()`는 버퍼 오버플로우 위험이 있어 실무에서는 `fgets()`를 사용한다.
> ```c
> fgets(name, sizeof(name), stdin);  // 안전한 대안
> ```

## 2. 진법 변환 (Number System Conversion)

### 진법의 종류

| 진법 | 기수 | 사용 숫자 | C 표기 |
|------|------|-----------|--------|
| 2진법 (Binary) | 2 | 0, 1 | `0b1010` |
| 8진법 (Octal) | 8 | 0 ~ 7 | `012` (앞에 0) |
| 10진법 (Decimal) | 10 | 0 ~ 9 | `10` |
| 16진법 (Hexadecimal) | 16 | 0 ~ 9, A ~ F | `0x1A` (앞에 0x) |

---

### 10진수 → 2진수 변환

10진수를 2로 나누면서 나머지를 **아래에서 위로** 읽는다.

```
13 ÷ 2 = 6 ... 1
 6 ÷ 2 = 3 ... 0
 3 ÷ 2 = 1 ... 1
 1 ÷ 2 = 0 ... 1
           ↑ 아래에서 위로 읽기
결과: 1101₂
```

---

### 2진수 → 10진수 변환

각 자리에 2의 거듭제곱을 곱해서 더한다.

```
1101₂ = 1×2³ + 1×2² + 0×2¹ + 1×2⁰
      = 8 + 4 + 0 + 1
      = 13
```

---

### 진법 변환 요약표

| 10진수 | 2진수 | 8진수 | 16진수 |
|--------|-------|-------|--------|
| 0 | 0000 | 0 | 0 |
| 1 | 0001 | 1 | 1 |
| 8 | 1000 | 10 | 8 |
| 10 | 1010 | 12 | A |
| 15 | 1111 | 17 | F |
| 16 | 10000 | 20 | 10 |
| 255 | 11111111 | 377 | FF |

---

### C언어에서 진법 출력

```c
#include <stdio.h>

int main() {
    int n = 255;
    printf("10진수: %d\n", n);   // 255
    printf("8진수:  %o\n", n);   // 377
    printf("16진수: %x\n", n);   // ff
    printf("16진수: %X\n", n);   // FF
    return 0;
}
```

## 3. 연산자 (Operator)

### (1) 산술 연산자 (Arithmetic Operator)

수학적 계산을 수행하는 연산자.

| 연산자 | 의미 | 예시 | 결과 |
|--------|------|------|------|
| `+` | 덧셈 | `5 + 3` | `8` |
| `-` | 뺄셈 | `5 - 3` | `2` |
| `*` | 곱셈 | `5 * 3` | `15` |
| `/` | 나눗셈 (몫) | `5 / 3` | `1` |
| `%` | 나머지 | `5 % 3` | `2` |

> 정수끼리 나누면 소수점이 버려진다. `5 / 2 = 2`  
> 실수 결과를 원하면 `5.0 / 2` 또는 `(float)5 / 2` 처럼 형변환 필요.

---

### (2) 관계 연산자 (Relational Operator)

두 값을 **비교**하는 연산자. 결과는 참(`1`) 또는 거짓(`0`).

| 연산자 | 의미 | 예시 | 결과 |
|--------|------|------|------|
| `==` | 같다 | `5 == 5` | `1` |
| `!=` | 다르다 | `5 != 3` | `1` |
| `>` | 크다 | `5 > 3` | `1` |
| `<` | 작다 | `5 < 3` | `0` |
| `>=` | 크거나 같다 | `5 >= 5` | `1` |
| `<=` | 작거나 같다 | `5 <= 3` | `0` |

---

### (3) 논리 연산자 (Logical Operator)

**조건을 조합**할 때 사용. 결과는 참(`1`) 또는 거짓(`0`).

| 연산자 | 의미 | 설명 | 예시 |
|--------|------|------|------|
| `&&` | AND (그리고) | 둘 다 참이어야 참 | `(5>3) && (2>1)` → `1` |
| `\|\|` | OR (또는) | 하나라도 참이면 참 | `(5>3) \|\| (2>5)` → `1` |
| `!` | NOT (부정) | 참↔거짓 반전 | `!(5>3)` → `0` |

---

### (4) 비트 연산자 (Bitwise Operator)

데이터를 **비트(0, 1) 단위**로 연산.

| 연산자 | 의미 | 예시 (5=0101, 3=0011) | 결과 |
|--------|------|----------------------|------|
| `&` | AND | `5 & 3` → `0101 & 0011` | `0001` = `1` |
| `\|` | OR | `5 \| 3` → `0101 \| 0011` | `0111` = `7` |
| `^` | XOR | `5 ^ 3` → `0101 ^ 0011` | `0110` = `6` |
| `~` | NOT (반전) | `~5` | 비트 전체 반전 |
| `<<` | 왼쪽 시프트 | `5 << 1` | `1010` = `10` (×2) |
| `>>` | 오른쪽 시프트 | `5 >> 1` | `0010` = `2` (÷2) |

---

### (5) 삼항 연산자 (Ternary Operator)

`if-else`를 한 줄로 표현하는 연산자.

```
조건 ? 참일 때 값 : 거짓일 때 값
```

```c
int a = 10, b = 20;
int max = (a > b) ? a : b;  // a > b가 거짓이므로 max = 20
printf("%d\n", max);        // 20
```

---

### (6) 대입 연산자 (Assignment Operator)

변수에 값을 **저장**하는 연산자.

| 연산자 | 의미 | 예시 | 동일 표현 |
|--------|------|------|-----------|
| `=` | 대입 | `a = 5` | - |
| `+=` | 더하고 대입 | `a += 3` | `a = a + 3` |
| `-=` | 빼고 대입 | `a -= 3` | `a = a - 3` |
| `*=` | 곱하고 대입 | `a *= 3` | `a = a * 3` |
| `/=` | 나누고 대입 | `a /= 3` | `a = a / 3` |
| `%=` | 나머지 대입 | `a %= 3` | `a = a % 3` |

---

### (7) 연산자 우선순위 (Operator Precedence)

우선순위가 높을수록 먼저 계산된다. 같은 순위는 **왼쪽에서 오른쪽**으로 계산.

| 순위 | 연산자 | 설명 |
|------|--------|------|
| 1 (최고) | `()` `[]` | 괄호, 배열 |
| 2 | `!` `~` `++` `--` | 단항 연산자 |
| 3 | `*` `/` `%` | 곱셈, 나눗셈, 나머지 |
| 4 | `+` `-` | 덧셈, 뺄셈 |
| 5 | `<<` `>>` | 비트 시프트 |
| 6 | `<` `<=` `>` `>=` | 관계 연산자 |
| 7 | `==` `!=` | 동등 비교 |
| 8 | `&` | 비트 AND |
| 9 | `^` | 비트 XOR |
| 10 | `\|` | 비트 OR |
| 11 | `&&` | 논리 AND |
| 12 | `\|\|` | 논리 OR |
| 13 | `? :` | 삼항 연산자 |
| 14 (최저) | `=` `+=` `-=` 등 | 대입 연산자 |

> 헷갈리면 무조건 **괄호 `()`** 를 써서 명시적으로 표현하는 것이 좋다.
