# C언어 자료형과 변수

---

## 1. 자료형 (Data Type)

데이터의 **종류와 크기**를 정의하는 것.  
변수에 저장할 수 있는 값의 형태를 결정한다.

| 자료형 | 크기 | 범위 | 설명 |
|--------|------|------|------|
| `char` | 1 byte | -128 ~ 127 | 문자 하나 |
| `int` | 4 byte | 약 -2,147,483,648 ~ 2,147,483,647 | 정수 |
| `float` | 4 byte | 소수점 약 6자리 | 단정밀도 실수 |
| `double` | 8 byte | 소수점 약 15자리 | 배정밀도 실수 |
| `void` | - | 없음 | 반환값 없음 |

> `unsigned`를 붙이면 음수 없이 양수 범위가 두 배로 늘어난다.  
> 예: `unsigned int` → 0 ~ 4,294,967,295

## 2. 변수 (Variable)

### 개념
데이터를 저장하는 **메모리 공간에 붙인 이름**.  
값을 저장하고, 꺼내 쓰고, 변경할 수 있다.

```c
int age = 20;       // int형 변수 age에 20 저장
char grade = 'A';   // char형 변수 grade에 문자 'A' 저장
float pi = 3.14;    // float형 변수 pi에 3.14 저장
```

---

### 변수명 작성 규칙

| 규칙 | 예시 |
|------|------|
| 영문자, 숫자, 언더스코어(`_`)만 사용 가능 | `my_var`, `count1` ✅ |
| **숫자로 시작 불가** | `1age` ❌ |
| **공백 불가** | `my age` ❌ |
| **대소문자 구분** | `Age`와 `age`는 다른 변수 |
| **예약어 사용 불가** | `int`, `for`, `while` 등 ❌ |

---

### C언어 예약어 (키워드)

C언어가 이미 특별한 용도로 사용하는 단어들로, 변수명으로 사용할 수 없다.

```
auto      break     case      char      const
continue  default   do        double    else
enum      extern    float     for       goto
if        int       long      register  return
short     signed    sizeof    static    struct
switch    typedef   union     unsigned  void
volatile  while
```

## 3. 변수의 종류

### (1) 지역변수 (Local Variable)

- 함수 **내부**에서 선언
- 선언된 함수 안에서만 접근 가능
- 함수 실행이 끝나면 **자동 소멸**
- 초기화하지 않으면 **쓰레기값(garbage value)** 이 들어있음

```c
void func() {
    int x = 10;  // 지역변수: func() 안에서만 사용 가능
    printf("%d\n", x);
}
// 여기서 x는 존재하지 않음
```

---

### (2) 전역변수 (Global Variable)

- 함수 **외부**에서 선언
- 프로그램 **어디서든** 접근 가능
- 프로그램이 종료될 때까지 메모리 유지
- 초기화하지 않으면 자동으로 **0**으로 초기화

```c
int count = 0;  // 전역변수

void add() {
    count++;
}

void print() {
    printf("%d\n", count);  // 다른 함수에서도 접근 가능
}
```

---

### (3) 정적변수 (Static Variable)

- `static` 키워드 사용
- 지역변수처럼 함수 안에서 선언하지만, 함수가 끝나도 **값이 유지됨**
- **최초 1회만 초기화**되고, 이후 호출에서는 이전 값을 이어받음
- 초기화하지 않으면 자동으로 **0**으로 초기화

```c
void counter() {
    static int count = 0;  // 처음 한 번만 초기화
    count++;
    printf("%d\n", count);
}

int main() {
    counter();  // 출력: 1
    counter();  // 출력: 2
    counter();  // 출력: 3
}
```

---

### (4) 동적변수 (Dynamic Variable)

- **힙(Heap) 메모리**에 프로그래머가 직접 할당하고 해제
- `malloc()` 으로 할당, `free()` 로 해제
- 해제하지 않으면 **메모리 누수(Memory Leak)** 발생
- `<stdlib.h>` 헤더 필요

```c
#include <stdlib.h>

int main() {
    int *p = (int*)malloc(sizeof(int));  // 메모리 할당
    if (p == NULL) return 1;            // 할당 실패 체크

    *p = 100;
    printf("%d\n", *p);

    free(p);   // 반드시 해제!
    p = NULL;  // 해제 후 NULL 대입 (안전)
}
```

---

### (5) 외부변수 (External Variable)

- 다른 파일에 선언된 전역변수를 **현재 파일에서 가져다 쓸 때** 사용
- `extern` 키워드 사용
- 선언만 하고 메모리는 원래 파일에서 관리

```c
// file1.c - 변수 정의
int count = 10;

// file2.c - 다른 파일의 변수 참조
extern int count;  // file1.c의 count를 가져다 씀

void print() {
    printf("%d\n", count);  // 10 출력
}
```

## 4. 변수 종류 한눈에 비교

| 종류 | 선언 위치 | 접근 범위 | 생존 기간 | 초기값 |
|------|-----------|-----------|-----------|--------|
| 지역변수 | 함수 내부 | 해당 함수 | 함수 실행 중 | 쓰레기값 |
| 전역변수 | 함수 외부 | 프로그램 전체 | 프로그램 전체 | 0 |
| 정적변수 | 함수 내부 (`static`) | 해당 함수 | 프로그램 전체 | 0 |
| 동적변수 | 힙 메모리 (`malloc`) | 포인터로 접근 | 직접 관리 | 쓰레기값 |
| 외부변수 | 다른 파일 (`extern`) | 프로그램 전체 | 프로그램 전체 | 0 |
