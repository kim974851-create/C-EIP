# 선행 처리기 (Preprocessor)

---

## 1. 개념

```
[ 내가 짠 코드 ]
      ↓
[ 선행 처리기 ]  ← 컴파일 전에 먼저 실행!
      ↓
[ 컴파일러 ]
      ↓
[ 실행 파일 ]
```

> 컴파일러가 코드를 읽기 **전에** 먼저 코드를 손질해주는 도구  
> `#` 으로 시작하는 줄을 처리함

---

## 2. 매크로 (Macro)

### 개념

> `#define` 을 사용해서 **특정 단어를 다른 값이나 코드로 바꿔치기** 해주는 것  
> 컴파일 전에 선행 처리기가 먼저 처리함  
> 세미콜론(`;`) 없음, 끝에 `;` 붙이면 `;`까지 치환되니 주의!

```
코드 작성 시        선행 처리기 처리 후
───────────────     ──────────────────
SQR(4)         →   (4 * 4)
PI             →   3.14
```

---

### 종류

#### 1) 매크로 상수 (Object-like Macro)

> **단순히 값을 이름으로 대체**하는 매크로

```c
#define 매크로이름 치환할값
```

```c
#define PI    3.14       // PI → 3.14
#define MAX   100        // MAX → 100
#define MSG   "Hello"    // MSG → "Hello"

int main() {
    printf("%f", PI);    // → printf("%f", 3.14)
    printf("%d", MAX);   // → printf("%d", 100)
    printf("%s", MSG);   // → printf("%s", "Hello")
}
```

| 특징 | 설명 |
|------|------|
| 인자 없음 | 괄호 `()` 사용 안 함 |
| 단순 치환 | 이름 → 값으로 그냥 교체 |
| 용도 | 상수값, 고정 문자열 정의 |

---

#### 2) 매크로 함수 (Function-like Macro)

> **인자(매개변수)를 받아서 처리**하는 매크로  
> 진짜 함수처럼 보이지만, 실제로는 **코드 바꿔치기**!

```c
#define 매크로이름(인자) 치환할식
```

```c
#define SQR(x)    (x * x)       // x를 제곱
#define ADD(a, b) (a + b)       // a와 b를 더함
#define MAX(a, b) ((a) > (b) ? (a) : (b))  // 더 큰 값

int main() {
    printf("%d", SQR(4));       // → (4 * 4)    = 16
    printf("%d", ADD(3, 5));    // → (3 + 5)    = 8
    printf("%d", MAX(3, 7));    // → (3) > (7) ? (3) : (7) = 7
}
```

| 특징 | 설명 |
|------|------|
| 인자 있음 | 괄호 `()` 안에 인자 선언 |
| 코드 치환 | 인자까지 통째로 바꿔치기 |
| 용도 | 간단한 연산, 반복 코드 줄이기 |

> ⚠️ **함정 주의!** 인자에 괄호를 안 쓰면 연산 순서가 달라짐!
> ```c
> #define SQR(x)   (x * x)      // 위험!
> SQR(3+1)  →  (3+1 * 3+1)  →  3 + 3 + 1 = 7  (예상: 16)
>
> #define SQR(x)   ((x) * (x))  // 안전!
> SQR(3+1)  →  ((3+1) * (3+1))  →  4 * 4 = 16  ✅
> ```

---

### 매크로 상수 vs 매크로 함수 비교

| 구분 | 매크로 상수 | 매크로 함수 |
|------|------------|------------|
| 인자 | 없음 | 있음 `(x)` |
| 예시 | `#define PI 3.14` | `#define SQR(x) (x*x)` |
| 용도 | 고정값 이름 붙이기 | 간단한 연산 처리 |
| 주의점 | 없음 | 인자에 괄호 필수! |

## 3. 주요 기능

---

### ① 매크로 처리 `#define`

```c
#define PI 3.14        // PI → 3.14 로 바꿔치기
#define SQR(x) (x*x)  // 함수처럼 쓰는 매크로

int main() {
    printf("%f", PI);        // → printf("%f", 3.14)
    printf("%d", SQR(4));    // → printf("%d", (4*4))
}
```

| 종류 | 예시 |
|------|------|
| 객체형 매크로 (단순 치환) | `#define PI 3.14` / `#define MAX 100` |
| 함수형 매크로 (인자 있음) | `#define SQR(x) (x*x)` / `#define ADD(a,b) (a+b)` |

---

### ② 파일 포함 `#include`

```c
#include <stdio.h>    // ← 시스템 헤더파일  (< > 사용)
#include "myfile.h"   // ← 내가 만든 파일   (" " 사용)
```

| 문법 | 대상 | 예시 |
|------|------|------|
| `< >` 사용 | 시스템 헤더파일 | `#include <stdio.h>` / `#include <math.h>` |
| `" "` 사용 | 내가 만든 헤더파일 | `#include "mymath.h"` / `#include "util.h"` |

> 해당 파일의 내용을 **그 자리에 통째로 붙여넣기** 해주는 것!

---

### ③ 조건부 컴파일 `#ifdef / #ifndef`

```c
#define DEBUG   // DEBUG 매크로 정의

#ifdef DEBUG              // DEBUG가 정의되어 있으면?
    printf("디버그 모드\n");   // ← 이 코드 포함
#else
    printf("배포 모드\n");    // ← 이 코드 제외
#endif                    // 조건 끝
```

| 지시문 | 의미 |
|--------|------|
| `#ifdef` | 매크로가 정의되어 있으면 컴파일 |
| `#ifndef` | 매크로가 정의되어 있지 않으면 컴파일 |
| `#if` | 조건식이 참이면 컴파일 |
| `#else` | 위 조건이 거짓이면 |
| `#elif` | else if 와 같음 |
| `#endif` | 조건부 컴파일 끝 |

---

### ④ 문자열 치환 `#define` + `#undef`

```c
#define MSG "Hello"    // MSG → "Hello" 로 치환
printf("%s", MSG);     // → printf("%s", "Hello")

#undef MSG             // MSG 정의 해제
// 이 아래부터 MSG 사용 불가!
```

| 지시문 | 의미 |
|--------|------|
| `#define` | 매크로(문자열) 정의 |
| `#undef` | 매크로 정의 해제 |

---

## 4. 한눈에 정리

| 기능 | 지시문 | 핵심 |
|------|--------|------|
| 매크로 처리 | `#define` | 코드 바꿔치기 |
| 파일 포함 | `#include` | 파일 붙여넣기 |
| 조건부 컴파일 | `#ifdef ~ #endif` | 조건에 따라 코드 선택 |
| 문자열 치환 | `#define` / `#undef` | 문자열 → 다른 값으로 |

## 5. 문제 풀이 예시

### 예제 1 - 매크로 치환 함정

```c
#include <stdio.h>
#define SQR(x) (x * x)   // SQR(x)를 (x * x)로 바꿔치기

int main() {
    printf("%d", SQR(3+1));
    //           ↓ 바꿔치기 발생!
    //           (3+1 * 3+1)
    //
    // 계산 순서 (곱셈이 덧셈보다 먼저!)
    // (3 + 1*3 + 1)
    //  3 +  3  + 1  = 7
    //
    // 출력 결과: 7
    return 0;
}
```

> 💡 `#define SQR(x) ((x) * (x))` 로 괄호를 감싸야 안전!

---

### 예제 2 - for + LIMIT

```c
#include <stdio.h>
#define LIMIT 5   // LIMIT를 숫자 5로 바꿔치기

int main(void) {
    int i, sum = 0;   // i = 반복용 숫자, sum = 합계 (초기값 0)

    for (i = 0; i < 5; i += 2) {
    //   ↑ 시작  ↑ 조건   ↑ 매 반복마다 i에 2씩 더함
        sum += i;   // sum = sum + i
    }

    // ┌──────┬─────┬──────────────┬─────┐
    // │ 회차 │  i  │   조건 i < 5 │ sum │
    // ├──────┼─────┼──────────────┼─────┤
    // │  1   │  0  │  0 < 5  ✅   │  0  │  sum = 0 + 0
    // │  2   │  2  │  2 < 5  ✅   │  2  │  sum = 0 + 2
    // │  3   │  4  │  4 < 5  ✅   │  6  │  sum = 2 + 4
    // │  4   │  6  │  6 < 5  ❌   │  6  │  조건 실패 → 탈출
    // └──────┴─────┴──────────────┴─────┘

    printf("%d\n", sum);   // 출력: 6
    return 0;
}
```

### 예제 3 - while + 조건 분기

```c
#include <stdio.h>
#define STEP 3   // STEP → 3으로 바꿔치기
#define END  10  // END  → 10으로 바꿔치기

int main(void) {
    int x = 0;

    while (x < 10) {   // x가 10보다 작으면 계속 반복
        x += 3;        // x에 3을 더함
        if (x % 2 == 0)   // x를 2로 나눈 나머지가 0이면? (짝수면?)
            x++;           // x에 1을 더함
    }

    // ┌──────┬─────────┬──────────┬──────────────┬──────────────┬─────┐
    // │ 회차 │ x < 10? │  x += 3  │ x % 2 == 0?  │     x++      │  x  │
    // ├──────┼─────────┼──────────┼──────────────┼──────────────┼─────┤
    // │  1   │ 0 < 10✅│  0+3= 3  │ 3%2=1  → ❌  │  (생략)      │  3  │
    // │  2   │ 3 < 10✅│  3+3= 6  │ 6%2=0  → ✅  │  6+1= 7      │  7  │
    // │  3   │ 7 < 10✅│  7+3=10  │ 10%2=0 → ✅  │  10+1=11     │ 11  │
    // │  4   │11 < 10❌│  (탈출)  │              │              │ 11  │
    // └──────┴─────────┴──────────┴──────────────┴──────────────┴─────┘

    printf("%d\n", x);   // 출력: 11
    return 0;
}
```

### 예제 4 - switch + 매크로

```c
#include <stdio.h>
#define LEVEL 2   // LEVEL → 2로 바꿔치기
#define BONUS 5   // BONUS → 5로 바꿔치기

int main(void) {
    int score = 0;
    int i;

    for (i = 0; i < 3; i++) {   // i = 0, 1, 2  (총 3번 반복)

        switch (LEVEL + i) {    // switch(2 + i) → i에 따라 값이 바뀜
            case 1:
                score += 1;     // 2+i == 1 일 때 (해당 없음)
                break;
            case 2:             // 2+i == 2 일 때
            case 3:             // 2+i == 3 일 때  ← case 2,3 묶음!
                score += 2;
                break;
            default:            // 나머지 전부
                score += 5;     // BONUS = 5
        }
    }

    // ┌──────┬─────┬────────────┬──────────┬───────────────────┬───────┐
    // │ 회차 │  i  │  2 + i     │ 해당case │      동작         │ score │
    // ├──────┼─────┼────────────┼──────────┼───────────────────┼───────┤
    // │  1   │  0  │ 2+0 = 2    │  case 2  │ score += 2 → 0+2  │   2   │
    // │  2   │  1  │ 2+1 = 3    │  case 3  │ score += 2 → 2+2  │   4   │
    // │  3   │  2  │ 2+2 = 4    │ default  │ score += 5 → 4+5  │   9   │
    // └──────┴─────┴────────────┴──────────┴───────────────────┴───────┘

    printf("%d\n", score);   // 출력: 9
    return 0;
}
```
